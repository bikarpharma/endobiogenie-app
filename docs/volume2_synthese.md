## 1. FORMULES INDEX BdF CRITIQUES (Volume 2)

### INDEX STARTER (Formule manquante du Volume 1)
```
Index Starter = IML / IMP
```
- **IML** = Index de Mobilisation des Leucocytes
- **IMP** = Index de Mobilisation des Plaquettes

### Index de Mobilisation des Leucocytes (IML)
- **Définition** : Capacité de libération des leucocytes séquestrés dans l'espace splanchnique vs hépatique
- **Facteur** : Alpha-sympathique adaptatif
- **Élevé** : Congestion vasculaire hépatique due à sollicitation du lit splanchnique
- **Bas** : Flux adaptatif hépatique → splanchnique

### Index de Mobilisation des Plaquettes (IMP)
- **Définition** : Capacité de libération des plaquettes séquestrées dans l'espace splanchnique vs splénique
- **Facteur** : Bêta-sympathique
- **Élevé** : Mobilisation splanchnique → splénique (bêta prédominant)
- **Bas** : Mobilisation splénique → splanchnique (bêta bloqué)

### TABLEAU D'INTERPRÉTATION IML/IMP

| IML | IMP | Interprétation |
|-----|-----|----------------|
| ↑ | ↑ | Augmentation activité sympathique générale sur lit splanchnique |
| ↑ | ↓ | Bêta retardé → favorise spasmophilie |
| ↓ | ↓ | Hyper alpha adaptatif + bêta bloqué → congestion hépato-splénique |
| ↓ | ↑ | (Non spécifié) |

### SPASMOPHILIE - Dysfonction Sympathique

| Type | IML | Histamine | IMP |
|------|-----|-----------|-----|
| Bêta retardé | ↓ | ↓ | ↓ |
| Bêta bloqué | ↑ ou ↓ | ↑ | ↓ |

## 2. INDEX BdF PAR AXE

### 2.1 AXE CORTICOTROPE (Chapitre 8)

| Index | Définition | Signification si élevé | Signification si bas |
|-------|------------|------------------------|----------------------|
| βMSH/αMSH | Activité relative βMSH vs αMSH en adaptation | Endorphines insuffisantes | GABA insuffisant |
| ACTH | Activité organo-métabolique ACTH sur cortex surrénalien | Degré d'adaptation | - |
| Adaptation | Activité adaptative ACTH/FSH | Risque atopique | Risque auto-immun |
| Cortisol | Activité tissulaire effective cortisol | (Non corrélé aux taux sériques) | - |
| Cortex surrénalien | Taux global activité surrénale | Rapport cortisol/CS optimal 2-3 | < 2 améliorer cortisol |
| Permissivité CS | Rôle permissif cortex surrénalien | Risque atopique si histamine + adaptation ↑ | - |
| Adaptation-Permissivité | Activité adaptative vs permissive | - | Si bas/négatif : capacité adaptative compromise |

**Ratio Cortisol/Cortex Surrénalien :**
- **Optimal** : 2-3:1
- **> 4** : Cortisol excessif aux dépens androgènes surrénaliens → TX : alpha-sympatholytiques
- **< 2** : Androgènes surrénaliens prédominants → TX : adaptogènes

### 2.2 AXE GONADOTROPE (Chapitre 9)

| Index | Définition | Intérêt |
|-------|------------|---------|
| Index génital | GR/GB = Androgènes tissulaires / Œstrogènes tissulaires | Évaluer troubles fonctionnels |
| Index génital corrigé | Index génital × Index Starter | Évaluer troubles structurels |
| Index œstrogènes | Activité endocrinométabolique œstrogènes | TSH comme facteur pro-anabolique |
| Index œstrogènes corrigé | Activité tissulaire endocrinométabolique | Régulation intégrité/croissance tissus |
| IROOT (Rendement œstrogènes organo-tissulaires) | Activité globale œstrogènes / stimulation FSH | Élevé : prolifération récepteurs œstrogènes |
| Androgènes génitaux | Activité tissulaire endocrinométabolique androgènes | Proportionnel aux autres pour texture tissus |

### 2.3 AXE THYRÉOTROPE (Chapitre 10)

| Index | Définition | Si élevé | Si bas |
|-------|------------|----------|--------|
| Génito-thyroïdien | Neutrophiles/Lymphocytes = Œstrogènes vs TSH | Favorise inflammation/auto-immunité | Favorise hyper-immunité/atopie |
| Thyroïde | LDH/CPK = Activité métabolique cellulaire T4/T3 | Activité métabolique thyroïde ↑ | Activité métabolique thyroïde ↓ |
| Rendement thyroïdien | Vitesse adaptation thyroïde à TSH | Risque inflammation muqueuse | Risque hypertrophie tissulaire |
| PTH | Activité endocrinométabolique parathyroïde | Thyroïde inefficace, PTH compense | Réduire activité thyroïde |
| Remodelage osseux | Degré renouvellement osseux pour adaptation | Os sur-sollicité | - |

### 2.4 AXE SOMATOTROPE (Chapitre 11)

| Index | Définition | Si élevé | Si bas |
|-------|------------|----------|--------|
| Score de croissance | Distribution GH/Prolactine/Somatostatine | Risque croissance adénoïdienne | Désynchronisation somatotrope |
| Prolactine | Activité prolactine dans relance 2ème boucle corticotrope | Action prolongée relance cortisol | Action brève |
| Somatostatine | Activité sur pancréas exocrine | Pancréas exocrine sur-sollicité | Activité insuffisante pancréas |
| Insuline | Sensibilité insuline sur membrane cellulaire | Hyperinsulinisme | Sensibilité membrane insuffisante |
| Résistance insuline | Résistance intracellulaire pendant adaptation | Risque insuffisance mitochondriale | Risque ↑ radicaux libres |
| Index de croissance | Activité facteurs de croissance intracellulaire | Rôle IGF-1 et autres facteurs | - |
| Adénose | Tendance à l'adénose | Terrain favorable croissances | Risque métastases |

## 3. SPASMOPHILIE - CLEF DES TROUBLES (Chapitre 12)

### Définition
Dysfonction de :
1. **Séquençage autonome** (PΣ → αΣ → βΣ)
2. **Gestion endocrinienne du calcium** pendant adaptation

### 5 Niveaux d'Analyse

| Niveau | Description |
|--------|-------------|
| 1. Cause (terrain précritique) | Mauvaise gestion latente du calcium libre (calcémie normale) |
| 2. Agent | Agresseurs variés |
| 3. Réponse (terrain critique) | Réponse neuroendocrine excède calcium disponible |
| 4. Mécanismes | Symptômes divers selon localisation |
| 5. Effets | Manifestations cliniques |

### LES 8 TYPES DE SPASMOPHILIE

| Type | Terrain | Traitement clé |
|------|---------|----------------|
| 1 | Femmes hyperfolliculiniques type hystéroïde | Inhiber FSH : Lithospermum officinale |
| 2 | Femmes hyperfolliculiniques hyperthyroïdiennes | + Inhiber thyroïde : Lycopus europaeus |
| 3 | Femmes androgéniques | Rééquilibrer rapport androgènes/œstrogènes |
| 4 | Hypoandrogénisme constitutionnel (hommes) | Relancer androgènes : Eleutherococcus, Satureja |
| 5 | Vagotonie enfance (1,5-6 ans) | Pancréas + α-sympatholytiques |
| 6 | Hyper-œstrogénisme/émancipation (28-32 ans) | ↓ TRH, ↓ Prolactine, drainage hépatique |
| 7 | Hypoandrogénisme adolescents | Harmoniseurs cortico-gonadotropes |
| 8 | Hypoandrogénisme adultes (gonadopause) | Relancer androgènes, ↓ SHBG |

### Signes Cliniques Spasmophilie

**Examen neurologique :**
- **Chvostek** : Spasmophilie neuromusculaire latente (percussion nerf facial)
- **Trousseau** : Spasmophilie vasculaire latente (main d'accoucheur)
- **Hippus** : Spasmophilie mentale (perception/conclusion) - spasme pupillaire
- **Réflexe glabellaire** : Spasmophilie mentale (conclusion/réaction)
- **Fasciculations langue** : Activation système réticulaire

## 4. INSUFFISANCE HÉPATIQUE (Chapitre 13)

### Concepts Clés
- **Foie** = Organe tampon central de l'endobiogénie
- L'insuffisance précède les anomalies histopathologiques flagrantes

### Index Hépatiques Critiques

| Index | Évalue | Interprétation |
|-------|--------|----------------|
| IML | Congestion splanchnique/hépatique | ↑ ou ↓ anormal = congestion |
| IMP | Congestion hépato-métabolique | Dépendance thrombopoïétine moelle |
| Index Starter (IML/IMP) | 3 niveaux fonction | Foie/Rate, Glucagon/Adrénaline, TRH-glucagon/alpha-cortisol |

### Principes de Drainage

**Ordre de priorité des émonctoires :**
1. **Foie** - Toujours en premier
2. **Rein**
3. **Lymphe**
4. **Intestin**
5. **Peau**

**Plantes drainantes principales :**
- **Hépatique** : Taraxacum officinale, Cynara scolymus
- **Hépatosplanchnique** : Carduus marianus
- **Hépatique basique** : Betula pubescens (Bgs)
- **Hépatorénal** : Zea mays Bgs

## 8. SECTION E - ALIMENTATION ENDOBIOGÉNIQUE

### Guide Général - Pyramide Alimentaire

| Fréquence | Aliments |
|-----------|----------|
| **Fréquent** | Eau de source, infusions, légumes (graines germées, amers, feuilles vertes), bouillons d'os, riz basmati/quinoa, poissons riches purines, laitages fermentés, baies |
| **Modéré** | Café/thé, noix, viandes nourries herbe, fromages durs ≥18 mois, alcool 3-4×/semaine |
| **Occasionnel** | Viandes saumurées 2-3×/mois, fromages cave |
| **Éviter** | Index glycémique élevé, gluten, graisses saturées, viandes antibiotiques/hormones/OGM |

### Régimes Spéciaux

| Régime | But | Index BdF cible |
|--------|-----|-----------------|
| **Alcalin** | Ramener pH neutre, ↑ capacité tampon | ↓ Cortisol, ↓ TSH, ↓ Insuline |
| **Épargne pancréas** | Réduire TSH, TRH, Prolactine | États inflammatoires, adénoïdiens, amyloïdes |
| **Pommes-sardines** | Spasmophilie, inflammation, toxines | Variable selon problème |
| **Riz brun** | Régulation sensibilité insuline | ↑/↓ Insuline, ↑/↓ Résistance insuline |

### Constipation - Approche Alimentaire

| Type | Approche |
|------|----------|

| Transit lent | Fibres insolubles (fruits peau, courges, légumes verts) |
