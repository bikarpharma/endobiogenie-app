## FONDEMENTS PHILOSOPHIQUES

### Definition de l'Endobiogenie
> "L'endobiogenie est une approche medicale qui considere l'organisme comme un systeme global regule par le systeme neuro-endocrinien."

### Les 5 Methodes Epistemologiques en Medecine

| Methode | Description | Limites |
|---------|-------------|---------|
| **Traditionalisme** | Sagesse ancestrale transmise | Dogmatisme, resistance au changement |
| **Empirisme** | Observation et experience | Manque de theorie unificatrice |
| **Rationalisme** | Logique deductive | Positivisme excessif, reductionnisme |
| **Intuition** | Connaissance directe | Subjectivite, non reproductible |
| **Synthetisme** | Integration de toutes les methodes | Complexite d'application |

### Concept de Terrain
Le terrain est la predisposition individuelle a developper certaines pathologies. Il est evalue par :
- L'interrogatoire clinique (symptomes, antecedents, phases de vie)
- La Biologie des Fonctions (index calcules depuis biomarqueurs standards)
- L'examen clinique oriente

### Holisme vs Reductionnisme
- **Reductionnisme** : 1 gene = 1 proteine = 1 maladie = 1 medicament
- **Holisme endobiogenique** : Systeme global neuro-endocrinien gerant le terrain

## SYSTEME NERVEUX AUTONOME

### Les 3 Branches du SNA

```
SNA
├── Parasympathique (πΣ) - Recuperation
│   ├── Autacoide : Serotonine (peripherique)
│   ├── Neurotransmetteur : Acetylcholine
│   └── Fonctions : Digestion, repos, regeneration
│
├── Alpha-sympathique (αΣ) - Congestion
│   ├── Autacoide : Histamine
│   ├── Neurotransmetteur : Adrenaline (alpha-recepteurs)
│   └── Fonctions : Vasoconstriction, congestion splanchnique
│
└── Beta-sympathique (βΣ) - Mobilisation
    ├── Autacoide : -
    ├── Neurotransmetteur : Adrenaline/Noradrenaline (beta-recepteurs)
    └── Fonctions : Energie, mobilisation, fight-or-flight
```

### Tableau Recapitulatif SNA

| Branche | Autacoide | Neurotransmetteur | Origine | Fonction |
|---------|-----------|-------------------|---------|----------|
| Parasympathique | Serotonine | Acetylcholine | Fibres peripheriques | Recuperation |
| Alpha-sympathique | Histamine | Adrenaline | Surrenales | Congestion |
| Beta-sympathique | - | Adrenaline/Noradrenaline | Surrenales | Mobilisation |

### Activite Centrale vs Peripherique de la Serotonine
- **Centrale** : Humeur, cognition, sommeil (cerveau)
- **Peripherique** : Motilite intestinale, hemostase (plaquettes = transporteur)

## LES 4 AXES ENDOCRINIENS

### Vue d'Ensemble

```
HYPOTHALAMUS → HYPOPHYSE → GLANDES PERIPHERIQUES → TISSUS

CORTICOTROPE : CRH → ACTH → Cortex surrenal → Cortisol/DHEA/Aldosterone
GONADOTROPE : GnRH → FSH/LH → Gonades → Oestrogenes/Progesterone/Androgenes
THYREOTROPE : TRH → TSH → Thyroide → T4/T3
SOMATOTROPE : GHRH → GH/PRL → Foie → IGF-1
```

### AXE CORTICOTROPE (Chapitre 6)

**Role** : Chef d'orchestre de l'ADAPTATION

#### Cascade Hormonale

```
Hypothalamus : CRH (Corticotropin-Releasing Hormone)
     ↓
Hypophyse : POMC → ACTH + β-endorphine + MSH
     ↓
Cortex surrenal :
├── Zona Fasciculata : CORTISOL (glucocorticoide)
├── Zona Reticularis : DHEA (androgene surrenalien)
└── Zona Glomerulosa : ALDOSTERONE (mineralocorticoide)
```

#### CORTISOL - L'Hormone de l'Adaptation

| Aspect | Effets |
|--------|--------|
| **Metabolique** | Gluconeogenese, lipolyse, catabolisme proteique |
| **Immunitaire** | Anti-inflammatoire, immunosuppresseur |
| **Cardiovasculaire** | Maintien pression arterielle, tonus vasculaire |
| **Osseux** | Inhibition osteoblastes, mobilisation calcium |
| **SNC** | Humeur, cognition, memoire |
| **Circadien** | Pic matinal (6-8h), nadir nocturne |

**Cortisol Permissif** : Permet l'action des autres hormones sans les remplacer

#### DHEA - L'Hormone de la Vitalite

| Phase de Vie | Role de la DHEA |
|--------------|-----------------|
| Foetogenese | Source d'oestrogenes placentaires |
| Enfance | Faible, augmente vers adrenarche |
| Puberte | Pic, maturation sexuelle |
| Age adulte | Maintien, declin progressif |
| Gonadopause | Compensation partielle du declin gonadique |

**Ratio Cortisol/DHEA** : Marqueur de l'equilibre catabolisme/anabolisme surrenalien

#### ALDOSTERONE

- **Fonction** : Regulation hydro-electrolytique (Na+/K+)
- **Systeme RAA** : Renine → Angiotensine I → Angiotensine II → Aldosterone
- **Effets** : Retention sodium, excretion potassium, volume plasmatique

### AXE GONADOTROPE (Chapitre 7)

**Role** : STRUCTURE et REPRODUCTION

#### Cascade Hormonale

```
Hypothalamus : GnRH (pulsatile)
     ↓
Hypophyse :
├── FSH (Follicle-Stimulating Hormone)
└── LH (Luteinizing Hormone)
     ↓
Gonades :
├── Ovaires : Oestrogenes, Progesterone
└── Testicules : Testosterone
     ↓
Tissus : Androgenes tissulaires (DHT, etc.)
```

#### FSH - Hormone Structurante

| Action | Description |
|--------|-------------|
| Fertilite | Maturation folliculaire, spermatogenese |
| Muqueuses | Hydratation, integrite epitheliale |
| Immunite | Modulation reponse immunitaire |
| Thyroide | Couplage gonado-thyreotrope |
| Morphologie | Structure tissulaire |

#### LH - Hormone de la Secretion

| Action | Description |
|--------|-------------|
| Ovulation | Declenchement |
| Testosterone | Stimulation cellules de Leydig |
| Progesterone | Corps jaune |
| Morphologie | Traits secondaires |

#### OESTROGENES

**3 Types principaux** :
1. **Estradiol (E2)** : Le plus puissant, pre-menopause
2. **Estrone (E1)** : Post-menopause, tissu adipeux
3. **Estriol (E3)** : Grossesse, placenta

| Effet | Description |
|-------|-------------|
| Anabolisme | Croissance tissulaire, proliferation |
| Vasculaire | Vasodilatation, protection cardiovasculaire |
| Osseux | Inhibition resorption osseuse |
| SNC | Neuroprotection, humeur |
| Immunitaire | Pro-inflammatoire (Th2) |

#### PROGESTERONE

| Effet | Description |
|-------|-------------|
| Uterus | Preparation implantation, maintien grossesse |
| SNC | Anxiolytique, sedatif (metabolites) |
| Temperature | Elevation phase luteale |
| Anti-oestrogene | Modulation effets oestrogenes |

#### ANDROGENES

**Testosterone** → converti en :
- **DHT** (5α-reductase) : Androgene tissulaire puissant
- **Estradiol** (aromatase) : Conversion peripherique

| Effet | Description |
|-------|-------------|
| Muscle | Anabolisme proteique |
| Os | Maintien masse osseuse |
| Libido | Desir sexuel (H et F) |
| Erythropoiese | Stimulation GR |
| SNC | Cognition, agressivite |

### AXE THYREOTROPE (Chapitre 8)

**Role** : METABOLISME ENERGETIQUE

#### Cascade Hormonale

```
Hypothalamus : TRH (Thyrotropin-Releasing Hormone)
     ↓
Hypophyse : TSH (Thyroid-Stimulating Hormone)
     ↓
Thyroide : T4 (Thyroxine) - pro-hormone
     ↓
Peripherie : T3 (Tri-iodothyronine) - hormone active

+ Glandes associees :
├── Parathyroides : PTH (Calcium ↑)
├── Cellules C thyroidiennes : Calcitonine (Calcium ↓)
└── Peau/Reins : Vitamine D3 active
```

#### TRH - Le Neuromodulateur Central

| Fonction | Description |
|----------|-------------|
| Adaptation creatrice | Reponse flexible aux changements |
| Chronobiologie | Rythmes circadiens |
| Vigilance | Intensite de l'eveil |
| SNA | Relance para-alpha-beta |
| Serotonine-Dopamine | Integration centrale |

#### TSH

- **Faible correlation avec fT4** : R² = 0.236 seulement !
- **Reflete** : Reactivite thyroidienne, pas efficacite finale
- **Usage BdF** : Index directs et indirects

#### T4 et T3

| Parametre | T4 | T3 |
|-----------|----|----|
| Type | Pro-hormone | Hormone active |
| Production | 100% thyroidienne | 20% thyroidienne, 80% peripherique |
| Demi-vie | 7 jours | 1 jour |
| Activite | Faible | 3-5x plus active |

**Conversion T4 → T3** :
- Deiodinase type 1 et 2 (activation)
- Deiodinase type 3 (inactivation → rT3)

#### Relation LDH/CPK et Fonction Thyroidienne

| Etat | Ratio LDH/CPK |
|------|---------------|
| Hyperthyroidie | 2.65 |
| Hyperthyroidie subclinique | 2.15 |
| Normal | 1.99 |
| Hypothyroidie subclinique | 1.89 |
| Hypothyroidie | 1.08 |

#### PTH, Calcitonine, Vitamine D3

```
CALCIUM BAS → PTH ↑ → Resorption osseuse ↑ + Absorption intestinale ↑ + Reabsorption renale ↑
CALCIUM HAUT → Calcitonine ↑ → Depot osseux ↑
VITAMINE D3 : Absorption intestinale calcium, mineralisation osseuse
```

#### THYMUS

- **Fonction** : Maturation lymphocytes T
- **Involution** : Progressive avec l'age (apres puberte)
- **Hormones thymiques** : Thymosine, thymuline

### AXE SOMATOTROPE (Chapitre 9)

**Role** : CROISSANCE et REPARATION

#### Cascade Hormonale

```
Hypothalamus :
├── GHRH (stimulant)
└── Somatostatine (inhibiteur)
     ↓
Hypophyse :
├── GH (Growth Hormone)
└── Prolactine
     ↓
Foie : IGF-1 (Insulin-like Growth Factor 1)
     ↓
Pancreas :
├── Insuline (anabolisant)
└── Glucagon (catabolisant)
```

#### GH - Hormone de Croissance

| Effet | Description |
|-------|-------------|
| Croissance | Elongation osseuse (enfant) |
| Metabolisme | Lipolyse, gluconeogenese |
| Muscle | Anabolisme proteique |
| Immunite | Stimulation |

**Secretion pulsatile** : Pics nocturnes (sommeil profond)

#### PROLACTINE

| Fonction | Description |
|----------|-------------|
| Lactation | Galactogenese et galactopoiese |
| Immunite | Stimulation |
| Angiogenese | Neovascularisation |
| Recepteurs oestrogenes | Multiplication |

#### IGF-1

- **Production** : Hepatique (stimulee par GH)
- **Effets** : Croissance tissulaire, anabolisme
- **Inhibition apoptose** : Survie cellulaire

#### INSULINE et GLUCAGON

| Hormone | Effet | Glycemie |
|---------|-------|----------|
| Insuline | Anabolisme, stockage glucose | ↓ |
| Glucagon | Catabolisme, liberation glucose | ↑ |

## AXES COUPLES (Chapitre 10)

### 3 Types d'Associations Inter-Axes

| Type | Definition | Exemple |
|------|------------|---------|
| **Couplage** | Axes adjacents, regulation croisee | Cortico-Gonadotrope, Gonado-Thyreotrope |
| **Liaison** | Anabolique-Anabolique | Gonado-Somatotrope |
| **Attelage** | Catabolique-Catabolique | Cortico-Thyreotrope |

### Couplage Cortico-Gonadotrope

```
1ere boucle central-central : CRH ↔ GnRH
1ere boucle central-peripherique : ACTH → Surrenales → DHEA
2eme boucle : Cortisol ↔ Oestrogenes (feedback croise)
```

**Dysendocrinisme** : Desequilibre entre axes couples

### Couplage Gonado-Thyreotrope

```
1ere boucle central-central : GnRH ↔ TRH
1ere boucle central-peripherique : FSH/LH → Gonades
2eme boucle central : TSH ↔ FSH
2eme boucle peripherique : T3/T4 ↔ Oestrogenes
```

**Index Genito-Thyroidien** = Neutrophiles/Lymphocytes

### Couplage Thyreo-Somatotrope

```
1ere boucle : TRH ↔ GHRH/Somatostatine
2eme boucle : T3/T4 ↔ GH/IGF-1
```

## SYNDROMES D'ADAPTATION (Chapitre 12)

### Capacite Tampon

**Definition** : Reserve fonctionnelle permettant d'absorber les stress sans decompensation

| Type | Elements |
|------|----------|
| **Structurelle** | Redondance organique, reserve anatomique |
| **Fonctionnelle** | Metabolites, hormones, elements immunitaires |

**Organe tampon principal** : FOIE
- Capacite tampon hepatique = reserve metabolique
- Saturation → debordement → maladie

### Systeme Splanchnique

- **30% du volume sanguin** au repos
- **Congestion splanchnique** : Role du alpha-sympathique
- **Lit splanchnique** : Reserve mobilisable

### 5 Types de Syndromes d'Adaptation

| Type | Duree | Mecanisme |
|------|-------|-----------|
| **Immediat** | Secondes-minutes | SNA, adrenaline |
| **Court terme** | Heures-jours | Cortisol, hormones stress |
| **Chronique** | Semaines-mois | Remodelage, compensation |
| **General (SGA)** | Variable | Depletion capacite tampon |
| **Chronobiologique** | Saisonnier | Adaptation aux rythmes |

### Selye vs Endobiogenie

| Aspect | Selye (SGA) | Endobiogenie |
|--------|-------------|--------------|
| Focus | Reponse au stress | Gestion du terrain |
| Phases | Alarme, Resistance, Epuisement | Adaptation continue |
| Agent | Stresseur externe | Terrain + Agent |
| Traitement | Supprimer le stresseur | Restaurer le terrain |

## LES 7 PHASES DE VIE (Chapitre 13)

### Vue d'Ensemble Chronologique

| Phase | Age | Axe Dominant | Caracteristiques |
|-------|-----|--------------|------------------|
| **Embryogenese** | 0-8 sem | Somatotrope | Organogenese |
| **Foetogenese** | 8-40 sem | Corticotrope | Maturation surrenalienne |
| **Petite enfance** | 0-11 mois | Thyreotrope tissulaire | Croissance rapide |
| **Enfance** | 1-11 ans | Thyreotrope | Developpement cognitif |
| **Adolescence** | 12-21 ans | Gonadotrope | Puberte, maturation sexuelle |
| **Age adulte** | 22-46 ans | Tous | Equilibre, reproduction |
| **Gonadopause** | 47+ | Corticotrope | Declin gonadique, recyclage |

### Programmes et Sous-Programmes Endocriniens

Chaque phase a :
1. **Programme principal** : Axe dominant
2. **Sous-programmes** : Sequences specifiques
3. **Transitions** : Periodes de vulnerabilite

### Autopathogenicite

**Definition** : Axe genetiquement le plus susceptible de dysfonctionner

- Determine par genetique + epigenetique
- Revele par histoire de vie
- Guide le traitement preventif

### 5 Stades de Manifestation de la Maladie

1. **Terrain precritique** : Predisposition latente
2. **Agent** : Declencheur (stress, infection, trauma)
3. **Terrain critique** : Decompensation
4. **Mecanisme** : Physiopathologie active
5. **Effet** : Symptomes cliniques

## BIOLOGIE DES FONCTIONS (Chapitre 15)

### Definition

> "Systeme de modelisation developpe par Dr Duraffourd qui simule l'activite biologique via les effets directs/indirects de l'activite neuroendocrinienne"

### Les 17 Biomarqueurs

| Origine | Biomarqueurs |
|---------|--------------|
| **Moelle osseuse** | GR, GB, Neutrophiles%, Lymphocytes%, Eosinophiles%, Monocytes%, Basophiles%, Hemoglobine |
| **Interaction moelle/serum** | VS |
| **Stroma osseux** | Osteocalcine, PAO isoenzyme osseuse |
| **Enzymes generales** | LDH, CPK |
| **Endocrinien** | TSH |
| **Electrolytes** | Potassium, Calcium |

### Reference Normative

**Femme adulte pre-menopausee** = Reference pour tous les index

### Signification des Biomarqueurs

| Biomarqueur | Marqueur de... |
|-------------|----------------|
| **GR** | Activite androgenique tissulaire |
| **GB** | Activite oestrogenique tissulaire |
| **Neutrophiles** | Oestrogenes dans immunite/inflammation/anabolisme |
| **Monocytes** | Insuffisance relative FSH→oestrogenes (↑ = dysadaptation) |
| **Eosinophiles** | Sollicitation ACTH (↑) / Insuffisance cortisol (↑) |
| **Lymphocytes** | Inversement ∝ cortisol et oestrogenes, directement ∝ besoins TSH |
| **Plaquettes** | Adaptation, transporteur serotonine |
| **Hemoglobine** | Intensite alpha-sympathique dans adaptation |
| **Osteocalcine** | Regulation energie (glucose, graisses, ATP) - impliquee dans >60% index |
| **CPK** | Renouvellement musculaire, transfert ADP↔ATP |
| **LDH** | Oxydation alteree glucose (pyruvate↔lactate) |
| **TSH** | Reactivite thyroide (R² = 0.236 avec fT4) |

### 3 Postulats Fondamentaux de la BdF

1. **Le systeme endocrinien est le gestionnaire du terrain**, le SNA le module
2. **Les biomarqueurs = production metabolique en aval** de la gestion neuroendocrinienne
3. **Les valeurs ne s'ecartent pas significativement des normes** (attention TSH < 0.5 ou > 5)

### Index Directs

| Index | Formule | Signification | Normes |
|-------|---------|---------------|--------|
| **Index Genital** | GR/GB | Androgenes/Oestrogenes tissulaires | 0.70-0.85 |
| **Index Genito-Thyroidien (GT)** | Neutro/Lympho | Couplage gonades/thyroide | 1.5-2.5 |
| **Index d'Adaptation** | Eosino/Mono | ACTH-cortisol vs FSH-oestrogenes | 0.25-0.50 |
| **IMP** | Plaq/(60×GR) | Capacite mobilisation adrenaline | 0.85-1.15 |
| **Index Oestrogenique** | TSH/Osteocalcine | Activite metabo-endocrinienne oestrogenes | Variable |
| **Index de Croissance** | PAOi/Osteocalcine | Activite metabolique GH | Variable |
| **Index Thyroidien** | LDH/CPK | Activite metabolique thyroide cellulaire | 3.5-5.5 |

### Index Indirects

| Index | Composition | Signification |
|-------|-------------|---------------|
| **Index Genital Corrige** | Index Genital × Index Starter | Equilibre reel androgenes/oestrogenes |
| **Index d'Histamine Evoquee** | (Eosino² × Plaq) / (Index Corticosurrenal × Mono) | Activite histaminique |
| **Index d'Apoptose** | Index Expansion Structurelle / Index Expansion Membranaire | Mort cellulaire programmee |
| **Index de Rendement Thyroidien** | Index Thyroidien / TSH | Efficacite thyroidienne reelle |
| **Index de Cortisol** | (Index Catabolisme/Anabolisme) / Index d'Adaptation | Activite cortisol |

### Valeurs Structure vs Fonction

- **Index Starter** permet de distinguer valeurs structurelles vs fonctionnelles
- La plupart des index ont les 2 dimensions

|----------|-------------------|----------|

### Extraits Globaux vs Composes Isoles

| Aspect | Extrait Global | Compose Isole |
|--------|----------------|---------------|
| Securite | Plus sur (antagonisme partiel) | Risque toxicite |
| Spectre | Large, polyvalent | Etroit, cible |
| Exemple | Eucalyptus HE (70% eucalyptol) | Eucalyptol 100% (toxique <2 ans) |

## THERAPEUTIQUE ENDOBIOGENIQUE (Chapitre 17)

### Principes Fondamentaux

1. **Ne pas nuire** (Primum non nocere)
2. **Dose la plus faible possible**
3. **Le moins longtemps possible**
4. **Restaurer le terrain**
5. **Soulager plus de souffrance que d'en causer**

### Objectif Therapeutique

> **Restitutio ad integrum** : Restaurer le corps dans son etat fonctionnel anterieur

### Spectre Therapeutique selon Capacite Tampon

```
Capacite tampon ELEVEE → Interventions non pharmaceutiques
├── Mode de vie
├── Alimentation
├── Oligoelements
├── Plantes medicinales
├── Vitamines, Antioxydants
└── Acides amines

Capacite tampon FAIBLE → Produits pharmaceutiques necessaires
```

## INDEX ALPHABETIQUE DES CONCEPTS

### A
- **Adaptation** : Capacite de l'organisme a maintenir l'homeostasie
- **ACTH** : Hormone adrenocorticotrope, stimule cortex surrenal
- **Aldosterone** : Mineralocorticoide, regulation Na+/K+
- **Androgenes** : Hormones masculinisantes (testosterone, DHT)
- **Anamnese** : Recueil systematique de l'histoire du patient
- **Autopathogenicite** : Axe genetiquement vulnerable

### B
- **BdF** : Biologie des Fonctions - systeme d'index calcules
- **Biomarqueur** : Element mesurable refletant une fonction biologique

### C
- **Capacite tampon** : Reserve fonctionnelle avant decompensation
- **Cortisol** : Glucocorticoide principal, hormone de l'adaptation
- **Couplage** : Association entre axes endocriniens adjacents

### D
- **DHEA** : Dehydroepiandrosterone, androgene surrenalien
- **Drainage** : Stimulation des emonctoires pour elimination

### E
- **Emonctoire** : Organe d'elimination (foie, reins, peau, poumons, intestins)
- **Endobiogenie** : Approche medicale basee sur le terrain neuro-endocrinien

### F
- **FSH** : Hormone folliculo-stimulante
- **Forme galenique** : Mode de preparation d'une plante medicinale

### G
- **GH** : Hormone de croissance
- **Gonadopause** : Declin de la fonction gonadique (menopause/andropause)

### H
- **Holisme** : Vision globale de l'organisme
- **Histamine** : Autacoide du systeme alpha-sympathique

### I
- **IGF-1** : Insulin-like Growth Factor 1
- **Index Genital** : GR/GB, equilibre androgenes/oestrogenes
- **IMP** : Index Mobilisation Plaquettes

### L
- **LH** : Hormone luteinisante

### M
- **Melatonine** : Hormone pineale, rythmes circadiens
- **MG** : Macerat Glycerine (gemmotherapie)

### O
- **Oestrogenes** : Hormones feminisantes
- **Oligoelements** : Mineraux en micro-doses

### P
- **POMC** : Pro-opiomelacortine, precurseur ACTH
- **Prolactine** : Hormone lactation et immunite
- **PTH** : Hormone parathyroidienne

### R
- **Restitutio ad integrum** : Objectif therapeutique de restauration

### S
- **SGA** : Syndrome General d'Adaptation (Selye)
- **SNA** : Systeme Nerveux Autonome
- **Spasmophilie** : Syndrome de dysfonction du SNA
- **Splanchnique** : Relatif aux visceres abdominaux

### T
- **Terrain** : Predisposition individuelle aux pathologies

- **TRH** : Hormone de liberation de la thyrotropine
- **TSH** : Hormone thyreostimulante
