## INDEX BdF - AXE GONADOTROPE (Chapitre 3)

### TABLE 3.12 - Index Centraux Gonadotropes

| Index | Définition | Import | Corrélations |
|-------|------------|--------|--------------|
| **FSH** | Activité organométabolique FSH | Évalue efficacité stimulation organe cible + feedback négatif | Direct: ACTH | Inverse: Œstrogènes tissulaires, Adaptation, Index LH |
| **LH** | Activité organométabolique LH | Évalue efficacité stimulation organe cible + feedback négatif | Direct: ACTH, Index FSH | Inverse: Progestérone, Androgènes tissulaires |

### TABLE 3.13 - Interprétations Index Pituitaires Élevés

| Situation | BdF | Traitement |
|-----------|-----|------------|
| Rendement organométabolique bas | ↑ FSH, ↓ OTEYi, ↓ Œstrogènes corrigés | Soutenir FSH: Inula helenium, Quercus pedunculata (indirect). Support œstrogènes: Angelica archangelica, Salvia officinalis, Salvia sclarea |
| Feedback négatif altéré | ↑ LH malgré androgènes élevés | Inhiber LH: Alchemilla vulgaris, Medicago sativa, Prunus africana. Inhiber androgènes: Fragaria vesca, Humulus lupulus, Menyanthes trifoliata |
| Feedback autre hormone pituitaire altéré | ↑ LH, ↑ Androgènes génitaux comparatifs | Inhiber LH + Inhiber FSH: Borago officinalis + Inhiber œstrogènes: Vitex agnus castus |

### TABLE 3.15 - Index Périphériques Gonadotropes

| Index | Définition | Import | Direct | Inverse |
|-------|------------|--------|--------|---------|
| **Genital Ratio (GR)** | Androgènes/Œstrogènes tissulaires | Évalue en adaptation aiguë | GR, LMI | PMI |
| **Genital Ratio Corrigé (GRc)** | Activité androgènes/œstrogènes hors adaptation | Évalue troubles structurels | - | - |
| **Index Œstrogènes** | Activité endocrinométabolique œstrogènes | - | TSH | Ostéocalcine, Activité nucléomembranaire |
| **Index Œstrogènes Corrigé** | Activité tissulaire endocrinométabolique | Évalue croissance tissulaire potentielle | Index Œstrogènes, OTEYi | - |
| **OTEYi** (Organotissular Estrogen Yield) | Activité organotissulaire œstrogènes / FSH | ↑ = prolifération récepteurs œstrogènes | WBC | Index FSH |
| **Androgènes Génitaux Corrigés** | Activité tissulaire androgènes gonadaux | Évalue croissance structurelle | Index Œstrogènes, Index Adaptation | GRc |

### TABLE 3.16 - Sites Production Œstrogènes

## INDEX BdF - AXE SOMATOTROPE (Chapitre 6)

### TABLE 6.15 - Index Centraux Somatotropes

| Index | Définition | Import | Direct | Inverse |
|-------|------------|--------|--------|---------|
| **GH Growth Score** | Activité endocrinométabolique GH | ↑: utilisation nutriments accrue, risque croissance adénoïdienne. ↓: désynchronisation somatotrope (hyperα/hyperthyroïdie centrale) | Score croissance | Score croissance corrigé, Index antigrowth, Index adénose |
| **Prolactine** | Niveau sollicitation SGA et modules systématisés | ↑: PL active relance cortisol (insuffisance dopamine). ↓: PL diminuée par somatostatine centrale et/ou non requise (qualité cortisol) | Somatostatine, Index GH | TSH, Index Cortisol |

### TABLE 6.16 - Index Périphériques Somatotropes

| Index | Définition | Import | Direct | Inverse |
|-------|------------|--------|--------|---------|
| **Somatostatine** | Activité somatostatine / pancréas exocrine | ↑: pancréas exocrine sur-sollicité, croissance hypertrophique. ↓: activité pancréas exocrine insuffisante, croissance hyperplasique | Index Antigrowth, Cortisol | - |
| **Insuline** | Activité endocrinométabolique insuline | ↑: hyperinsulinisme, risque désynchronisation somatotrope. ↓: hyperinsulinisme par sensibilité membranaire insuffisante | Catabolisme/Anabolisme, Résistance insuline | Redox, Radicaux libres nocifs |
| **Résistance Insuline** | Inhibition insuline au niveau membranaire | ↑: favorise activité 1ère boucle prolongée, nutrition organes vitaux aux dépens autres. ↓: distribution glucose compromise organes vitaux | Insuline | Redox, Radicaux libres nocifs |
| **Index Croissance** | Activité intracellulaire facteurs croissance (IGF-1) | Évalue IGF-1 et autres facteurs croissance | Phosphatase alcaline osseuse | Ostéocalcine, Antigrowth, Démyélinisation, Expansion membranaire |

### TABLE 6.17 - Effets Métaboliques Généraux

| Index | Définition | Import | Direct | Inverse |
|-------|------------|--------|--------|---------|
| **Catabolisme** | Niveau activité catabolique | Catabolisme nourrit anabolisme | Index Thyroïde, Index Cortex Surrénalien | Index Génito-thyroïdien, Catabolisme-Anabolisme |
| **Anabolisme** | Niveau activité anabolique | Anabolisme assure restauration organisme | Catabolisme | Catabolisme-Anabolisme |
| **Rendement Métabolique** | Taux métabolisme global | Exprime efficacité générale production/répartition | Catabolisme, Anabolisme | Ischémie, Fracture membranaire |
| **Ischémie** | Congestion tissulaire / activité métabolique | - | Remodelage osseux | Rendement métabolique, Congestion splanchnique |
| **Index Démyélinisation** | Adaptabilité réponse énergético-métabolique insuline | ↑: désynchronisation somatotrope, insuline précède facteurs croissance | Insuline, Index Somatostatine corrigé | Index Croissance, Index Amyloïde |
| **Index Adénose** | Activité facteurs endocrines propices hyperplasie | ↑: favorise terrain croissances adénoïdiennes | Index Ostéocalcine, Index TRH/TSH | Rendement ischémie |

### TABLE 6.18 - Activité Métabolique Cellulaire

| Index | Définition | Import | Direct | Inverse |
|-------|------------|--------|--------|---------|
| **Perméabilité Cellulaire Active** | Activité dynamique nutrition transmembranaire | ↑: favorise 1ère boucle, perméabilité membranaire | TSH, Insuline | Somatostatine |
| **Perméabilité Cellulaire Passive** | Permissivité osmotique transmembranaire | ↓: fluidité membranaire insuffisante | Nécrose | Pas de dénominateur = non-adaptation |
| **Redox** | Oxydoréduction globale (réduction + oxydation glucose/lipides pour ATP) | ↓: immunité compromise, risque insuffisance mitochondriale | Index Insuline | Index Somatostatine |
| **Radicaux Libres Nocifs** | Taux global radicaux libres circulants | - | Index Oxydoréduction | Fracture ADN |
| **Index Pro-Amyloïde** | Hypométabolisme intracellulaire | ↑: favorise insuffisance mitochondriale | Résistance Insuline | Index Réduction |

## MÉTRORRAGIES - 7 TYPES (Chapitre 5)

### MÉTRORRAGIES FOLLICULAIRES (Types 1-3)

#### Type 1A : ↑ Alpha-Sympathique

| Axe | Index | Valeurs |
|-----|-------|---------|
| Corticotrope | ACTH | ↓ ou normal |
| | Cortisol | ↑ ou normal (cortisol/CS > 4) |

**TX :** Passiflora incarnata MT 60 mL, Rhodiola rosea MT 40 mL, Sequoia gigantea GM 80 mL, Quercus pedunculata GM 60 mL + Lavandula angustifolia EO 3 mL

### Plantes par Action Principale

#### SPASMOLYTIQUES
- **Valeriana officinalis** : #1 spasmophilie aiguë et chronique. GABAergique, Sérotoninergique, Mélatoninergique
- **Angelica archangelica** : Spasmolytique digestif/urinaire/neuromusculaire. Patients vagotoniques
- **Anthemis nobilis** (Camomille romaine) : Régulateur neuro-autonome. Troubles GI spasmophiliques
- **Tilia tomentosa** : Régulation douce, clearance toxines. Spasmophilies enfance

#### GONADOTROPES
- **Achillea millefolium** : Progestérone-like sans inhiber LH. Pelvien féminin
- **Alchemilla vulgaris** : Inhibition secondaire LH, maturation follicules. Génito-intestinal lutéal
- **Vitex agnus castus** : Régulateur dopaminergique pituitaire. PMS, hyperprolactinémie
- **Menyanthes trifoliata** : β-sympathomimétique, œstrogénique, FORTEMENT antiandrogénique

#### CORTICOTROPES
- **Eleutherococcus senticosus** : Adaptogène favorisant compétence cortico-androgénique
- **Rhodiola rosea** : Adaptogène, support surrénale
- **Sequoia gigantea** : Stimulant cortex surrénalien favorisant androgènes surrénaliens

#### THYROTROPES
- **Avena sativa** : Harmonise relation œstro-thyrotrope. Support pancréas exocrine
- **Lycopus europaeus** : Inhibiteur thyroïdien. Hyperthyroïdie
- **Melissa officinalis** : Adaptogène thyrotrope

#### DRAINAGE
- **Taraxacum officinale** : Draineur superbe. Hépatobiliaire, rénal. Conditions dysmetaboliques
- **Arctium lappa** : Draineur polyvalent dépuratif. Immuno-dermatologique
- **Carduus marianus** : Hépatoprotecteur/régénérateur

#### ANTI-INFLAMMATOIRES
- **Ribes nigrum** : Support général cortex surrénalien. Antiallergique
- **Urtica dioica feuille** : Antiallergique, revitalisant tissulaire
- **Urtica dioica racine** : Bloque facteurs croissance prostate, inhibe aromatase et 5α-réductase
